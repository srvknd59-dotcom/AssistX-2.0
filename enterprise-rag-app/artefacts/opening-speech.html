1 · The hook

"Before we touch a single line of code, I want to show you something."

[Open the chat UI. Let her type 2-3 questions herself. Don't explain anything yet — just watch.]

2 · Why this is different

"This is a chatbot. But it's not like the ones you've used before — it doesn't just make things up from whatever it remembers off the internet. It only answers using documents we hand it, and every single time, it shows you exactly which document — which sentence, even — it got that answer from."

Let that land
3 · Why it matters

"Companies everywhere are trying to build exactly this — a way for people to ask a question and get a trustworthy answer instead of digging through PDFs and old emails. It's called RAG, retrieval-augmented generation, and it's one of the most in-demand skills in tech right now."

4 · What today actually is

"By the end of today, you will have built this. Not a toy someone else built for you to click through — you're going to install a real database, load real documents into it, connect it to a chat model, and build the interface yourself. All running on this laptop, all yours."

5 · How today works

"I'm going to explain things, but you're driving the keyboard. Something is going to break today — that's not a maybe. When it does, we're not going to panic or skip past it. We're going to read the error message together and figure out what it's actually telling us, because that's the real skill, not memorizing commands."

"There's no wrong pace. If something doesn't click, we stay there. If we run behind, we cut the extra exercises at the end, never the fundamentals."

6 · Transition into Hour 1

"Ready? Let's start with what's actually happening inside that chat box you just used."
